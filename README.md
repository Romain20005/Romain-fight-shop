# Beschrijving
Voor dit project heb ik een fictieve webshop gemaakt rond vechtsportmateriaal.
Ik heb eerst een concept gekozen dat aansluit bij mijn persoonlijke interesse in MMA en grappling.
Daarna heb ik een eenvoudige structuur van pagina’s uitgetekend:
- Homepagina met overzicht van de producten
- Zes productdetailpagina’s
- About-sectie
- Contact-sectie
- Admin-gedeelte (login, dashboard, product-formulier)
